Скачать репозиторию через команду

    ```
    git clone https://github.com/janklod84/yii2-binance-project.git
    ```

Установить данный проект через команду:
```
composer install
```

Далее запустить сервер на порт 8080 через команду:
```
php yii serve
```

После этого ваш сайт будет доступен по ссылке:
```
http://localhost:8080
```

Можно выполнить короткую команду :
```
php server [ тогда приложение запустится также на порт 8080 по умолчанию ]
```

Тогда сайт будет доступен также по данной ссылке:
```
http://localhost:8080
```

Можно запусить сервер с указанного порта:
```
php server [тут_ваш_порт]
например: php server 8000
Тогда сайт будет доступен по указанному порту т.е по ссылке:
http://localhost:8000
```
